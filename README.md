# Inline Translator

Модуль для Foundry VTT, который выводит отыгрыш на новый уровень погружения. Он позволяет писать сообщения на различных языках D&D: персонажи, знающие язык, увидят понятный текст и его оригинальное "звучание", а те, кто не знает — увидят загадочные руны, символы или неразборчивый текст в зависимости от настроек Мастера.

## Установка
1. Используйте следующую ссылку манифеста для установки модуля.

    ```text
   https://raw.githubusercontent.com/nilinex/Inline/refs/heads/main/module.json
Тестировалось на версии 13.350, в системе DnD 5e 5.2.5. При обнаружении багов, просьба сообщать в ТГ.

2. Включите модуль в управлении модулями вашего мира.

## Использование в игре
Удостовериться, что в настройках участника выбран персонаж игкрока. 

Использовать следующий синтаксис:

     ~(Название языка)Текст~

Работает в описании предметов, чате, жургалах и может быть где-то еще...

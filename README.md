За основу идеи был взят модуль [Inline Table Rolls](https://github.com/spaceman023/Inline-Table-Rolls), переработан и изменен.

Ссылка для установки на Foundry VTT: https://github.com/nilinex/Inline/blob/main/module.json<br>
Тестировалось на версии 13.350, в системе DnD 5e 5.2.5.<br>
При обнаружении багов, просьба сообщать мне в ТГ.<br>

Модуль используется для реализации двух сценариев:
  1. Когда необходимо совершить бросок по случайной таблице,
  2. Когда персонаж пытается прочесть текст на незнакомом ему языке.

Оба сценария работают при использовании кода в чате, описании объектов и журналах.

БРОСОК ПО СЛУЧАЙНОЙ ТАБЛИЦЕ<br>
Необходимо использовать следующий синтаксис: <br>
^^#?Количество бросков по таблице?Наименование таблицы, по которой будет совершен бросок^^

Примеры использования синтаксиса:<br>
^^#?5?Монстры^^    - будет совершено 5 бросков по таблице "Монстры", будет отображено 5 случайных сторк из таблицы.<br>
^^#?1d4?Монстры^^  - будет совершено от 1 до 4х бросков по таблице "Монстры", и, согласно выпавшему значению, будет отображено несколько строк.<br>
^^#Монстры^^       - Будет совершен 1 бросок по таблице, а его результат отображен.<br>

Примеры использования сценария:<br>
Если написать в чат ^^#?1d4?Монстры^^, а в таблице указать ссылки на актеров, можно получить случайный набор актеров для сцены.

ИСПОЛЬЗОВАНИЕ ЯЗЫКОВ В ИГРЕ<br>
Необходимо использовать следующий синтаксис: <br>
\~\~(Название языка)Текст\~\~<br>
В настройках мастеру доступен выбор, как именно будет заменен текст для тех, кто язык не знает:<br>
<img width="533" height="113" alt="image" src="https://github.com/user-attachments/assets/bc3123be-0666-4336-86e8-5699c8835fcc" /><br>
По умолчанию доступна замена на случайные символы или на символ "А". (Планируется добавление вариантов замены в следующих версиях модуля)


Примеры использования синтаксиса:<br>
\~\~(Драконий)Тут хранятся сокровища!\~\~ - Этот текст увидят только те, кто владеет Драконьим.
\~\~(Всеобщий)Тут хранятся сокровища!\~\~ - Этот текст увидят все герои.

Примеры использования сценария:<br>
Если просто отправить в чат \~\~(Драконий)Тут хранятся сокровища!\~\~, то у того, кто не знает язык, все символы заменятся на случайные(согласно настройкам).<br>
